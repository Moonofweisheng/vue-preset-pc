# 魔法校前端 Preset PC

## Usage

```bash
vue create --preset savokiss/vue-preset-pc <project-name>
```
注意修改上面的 `<project-name>` 为要创建的文件夹名称


## Add Plugin

### 安装 Element-UI
```bash
vue add element
```

### 安装 Ant-Design-Vue
```bash
vue add ant-design
```

## 安装 Electron-Builder
```bash
vue add electron-builder
```
